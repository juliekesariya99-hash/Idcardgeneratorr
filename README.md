# 🪪 ID Card Generator

A simple and efficient application built using .NET that allows users to generate clean and professional identity cards with personal details and photos.

---

## 🚀 Features

* Generate ID cards instantly
* Add user details (Name, ID, Address)
* Upload and display profile photo
* Clean and structured card layout
* Print-ready output
* Lightweight and easy to use

---

## 🛠️ Technologies Used

* .NET Framework
* C#
* Windows Forms (WinForms)

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/idcardgeneratorr.git
   ```

2. Open the project in Visual Studio

3. Build the solution

4. Run the application

---

## ▶️ Usage

1. Enter user details:

   * Name
   * ID Number
   * Address

2. Upload a photo

3. Click on **Generate ID Card**

4. Preview and print the generated card

---

## 🎯 Use Cases

* Student ID cards
